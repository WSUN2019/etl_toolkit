A **Streamlit-based tool** to map and analyze folder structures. Easily generates a folder tree, statistics, and a preview of your files.  

---

## :gear: How to Run

1. **Open Command Prompt** and navigate to the project folder:

    ```cmd
    cd "C:\Users\[username]\python dev\etl_toolkit\FolderTools"
    ```

2. **Install requirements**
    ```cmd
    python -m pip install -r requirements.txt
    ```
    
3.  **Run Streamlit**
    ```cmd
    python -m streamlit run folder_toolbox.py
    ```
